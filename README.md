# Conference 

This simple project example how to work with ElasticSearch and ElasticSearch aggregations

* [Getting Started](#getting-started)
* [Used technologies](#used-technologies)

## Getting Started

You must to build spring-boot-application using the following command :

```bash
./mvn clean package 
```

## Used technologies

##### For back-end:

* [Spring Boot](https://spring.io/projects/spring-boot) 
* [Java](https://www.oracle.com/ru/java/technologies/)
* [Maven](https://maven.apache.org/)

##### For database connectivity:

* [H2](https://www.h2database.com/html/main.htmlc)
